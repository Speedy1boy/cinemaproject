# Система бронирования билетов в кинотеатр

Клиент-серверное веб-приложение для просмотра афиши кинотеатра, расписания сеансов и покупки билетов. Разработано в рамках курсового проекта (Naumen).

Проект представляет собой монорепозиторий, разделенный на клиентскую (SPA) и серверную (REST API) части. Реализована ролевая модель (Пользователь / Администратор), изоляция сессий через JWT и защита от параллельного бронирования одних и тех же мест.

## Структура репозитория
* `/cinema-backend` — исходный код серверной части (Java / Spring Boot).
* `/cinema-frontend` — исходный код клиентской части (React / Vite).
* `docker-compose.yaml` — конфигурация для развертывания инфраструктуры.

## Быстрый запуск (через Docker)
Для запуска всего проекта (БД + Бэкенд + Фронтенд) необходим установленный Docker.

В корневой директории выполните:
```bash
docker-compose up --build -d
```

После завершения сборки сервисы будут доступны по адресам:
* Клиентское приложение: `http://localhost:3001`
* Серверное API: `http://localhost:8081`
* База данных (PostgreSQL): `localhost:5432`

Для остановки контейнеров:
```bash
docker-compose down
```
